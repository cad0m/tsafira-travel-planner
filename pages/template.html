Page Title

This is a page template demonstrating dark mode implementation with CSS variables and proper class use throughout the HTML.

Content Section Title

This content section demonstrates proper dark mode styling with variable-based CSS.

Notice how we use:

  • CSS variables for colors: var(--color-bg), var(--color-text), etc.
  • Tailwind's dark mode classes: dark:bg-dark-bg
  • Custom CSS variables defined in main.css that respond to the .dark class

Feature One

Use var(--color-text-muted) for secondary text that adapts to dark mode.

Feature Two

All colors respond to the .dark class applied to the HTML element.
